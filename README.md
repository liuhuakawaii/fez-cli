## 特性 ✨

- 🚀 快速创建 React/Vue 项目
- 📦 内置多种项目模板
- 🎨 支持自定义模板
- 💡 交互式命令行体验
- 🔧 零配置，开箱即用

## 安装 📦
```bash
npm install -g fez-cli

yarn global add fez-cli
```

## 使用方法 🔨

# 创建新项目
```bash
fez create my-project
```

# 查看帮助
```bash
fez --help
```

## 可用模板 📋

- `react-ts` - React + TypeScript 模板
- `react-js` - React JavaScript 模板
- `vue-ts` - Vue3 + TypeScript 模板
- `vue-js` - Vue3 JavaScript 模板

## 配置项 ⚙️

在项目根目录创建 `.fezrc.js` 文件：

```javascript
module.exports = {
  // 配置项
}
```
